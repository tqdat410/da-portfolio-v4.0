## Overview

Hengout is a startup product focused on location discovery for GenZ users. It combines AI-based recommendation logic, personalization, and a group decision mode to suggest places that work for multiple people in one flow.

## Key Features

- Personalized venue recommendations
- Group consensus suggestion mode
- Event-driven activity updates through Kafka
- Redis caching for location and feed performance
- Apple In-App Purchase integration
- Monitoring dashboards with Grafana

## Role

Tech Lead in a 6-member team (3 engineering, 3 marketing). I owned architecture decisions and delivered core backend + deployment workflows.

## Responsibilities

- Designed microservices architecture
- Built backend services with Spring Boot
- Set up CI/CD and VPS deployment flow
- Integrated AI recommendation pipeline
- Drove payment integration and App Store release
- Implemented operational monitoring
