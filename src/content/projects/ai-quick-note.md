## Overview

Android note-taking app that supports multiple input formats and AI-assisted organization. It adds semantic retrieval features so users can query their notes through an intelligent chatbot flow.

## Key Features

- Multi-format note input (text, voice, image)
- Automatic AI-based categorization
- RAG-style chatbot retrieval
- Semantic search across stored notes
- Real-time synchronization via WebSocket
