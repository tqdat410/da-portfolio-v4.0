## Overview

Management platform for a Koi veterinary center, covering service browsing, appointment booking, billing, staff scheduling, and treatment record tracking.

## Key Features

- Service catalog and booking flow
- Billing and payment processing
- Medical records and treatment history
- Staff schedule management
- Admin monitoring tools

## Role

Team Leader and Backend Lead in a 5-member team.

## Responsibilities

- Defined architecture and delivery scope
- Coordinated task assignment
- Built and reviewed backend APIs
- Tracked progress and integration milestones

## Tech Stack

- ReactJS
- Spring Boot
- MySQL
- GitHub Actions
- Cloudinary
- AWS

## Key Learnings

Improved system design and backend execution in a monolith setup, while strengthening team coordination and Git-based collaboration practices.
