## Overview

Event management and registration platform for university communities. The system supports event publishing, attendee registration, QR-based attendance, and post-event survey workflows.

## Key Features

- Event creation and publishing pipeline
- Registration and attendance tracking
- QR check-in and check-out
- Survey builder and post-event feedback flow
- Result export for analysis

## Role

Team Leader and Fullstack Developer in a 5-member team.

## Responsibilities

- Owned architecture and delivery plan
- Developed Spring Boot backend services
- Set up CI/CD and Dockerized deployment
- Coordinated team execution and reviews

## Tech Stack

- Spring Boot
- Next.js
- PostgreSQL
- Cloudinary
- Docker
- VPS
- CI/CD

## Key Learnings

Built practical experience with production-like deployment pipelines, database design in PostgreSQL, and leadership under deadline-driven team delivery.
