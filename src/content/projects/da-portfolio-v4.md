## Overview

This portfolio is both a professional profile and an experimentation lab. It evolves as my skills grow, with a strong focus on interactive frontend techniques and design-driven storytelling.

## Key Features

- Interactive visual effects and 3D components
- Responsive and accessible layout patterns
- Structured project and profile presentation
- Performance-aware implementation
- Continuous iteration workflow

## Role

Owner and solo developer.

## Responsibilities

- Product concept and information architecture
- UI design and frontend implementation
- 3D integration and interaction experiments
- Performance tuning and deployment maintenance

## Architecture

Built with Next.js and TypeScript, styled with Tailwind CSS, and extended with Three.js via React Three Fiber for immersive visuals.

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Three.js
- React Three Fiber
- Framer Motion

## Results

- Delivered and maintained 4 major portfolio versions
- Built a single source of truth for my professional profile
- Expanded practical skills in WebGL-centric frontend engineering

## Key Learnings

Improved design-to-code execution and learned how to use modern AI coding tools to speed up iteration while keeping quality stable.
