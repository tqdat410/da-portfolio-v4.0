## Overview

Capstone graduation project building a centralized notification center inside SAP S/4HANA. The product includes a bell icon with badge count, dropdown notification list, read state controls, archive and delete actions, and subscription settings per user.

## Key Features

- Bell icon with badge and dropdown list
- Mark read/unread, archive, and delete actions
- Subscription settings and delivery channel preferences
- ABAP API for creating notifications from backend programs
- Real-time update flow via AMC/APC

## Role

Team Leader (4 members). I led requirement analysis, architecture direction, task coordination, quality review, and delivery reporting under enterprise process constraints.

## Responsibilities

- Requirement analysis and technical research
- Architecture design and integration planning
- Task assignment and progress tracking
- Documentation, test planning, and reporting

## Architecture

Built with SAP RAP and OData services on S/4HANA, Fiori/UI5 for user-facing interactions, background jobs for scheduled tasks, and AMC/APC channels for near real-time notification delivery.

## Tech Stack

- ABAP
- SAP RAP
- OData
- Fiori
- UI5
- HANA
- RESTful API
- AMC/APC
- Background Job
