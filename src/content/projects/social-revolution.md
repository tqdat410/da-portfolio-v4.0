## Overview

Educational simulation project for a Development Philosophy course, focused on socio-economic formation stages and the dynamics between them. Built as a solo project and adopted for course use.

## Key Features

- Stage-based socio-economic simulation
- Narrative-driven educational interaction
- AI-supported research and production flow

## Role

Owner and solo developer.

## Responsibilities

- Designed game logic and learning flow
- Wrote content scripts
- Implemented and deployed full application

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Cloudflare

## Results

- Strong lecturer feedback
- Reused as official learning material

## Key Learnings

Improved educational product design and gained more confidence combining AI-assisted workflow with interactive web engineering.
